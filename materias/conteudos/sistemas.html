Equação Linear

Uma equação linear é toda equação que pode ser escrita como:

\(a_1x_1 + a_2x_2 + ... + a_nx_n = b\)

a₁, a₂... são coeficientes; b é termo independente; x₁, x₂... incógnitas.
Exemplo: 2x₁ - 3x₂ + x₃ = 5

O que são Sistemas Lineares?

Um sistema linear é um conjunto de duas ou mais equações lineares com as mesmas incógnitas. Por exemplo:

\(\begin{cases} x + y = 14 \\ 4x + 2y = 48 \end{cases}\)

A solução de um sistema é o conjunto de valores das incógnitas que satisfaz todas as equações simultaneamente.

Classificação dos sistemas:

  • SPD (Sistema Possível e Determinado): possui uma única solução.
  • SPI (Sistema Possível e Indeterminado): possui infinitas soluções.
  • SI (Sistema Impossível): não possui solução.

Geometricamente, cada equação representa uma reta (2D) ou um plano (3D). O ponto ou região onde todas se intersectam é a solução do sistema.

Sistemas 2x2

Exemplo: Estacionamento com carros e motos

x = carros, y = motos
\(\begin{cases} x + y = 14 \\ 4x + 2y = 48 \end{cases}\)
x = 10, y = 4

Método da Substituição

Isolamos uma incógnita e substituímos na outra equação:

Sistema: { 3x - y = -11, x + 2y = 8 }
x = -2, y = 5

Método da Adição

Somamos ou subtraímos equações para eliminar uma incógnita:

Sistema: { x + y = 14, 4x + 2y = 48 }
x = 10, y = 4

Regra de Cramer

Método baseado em determinantes para sistemas quadrados:

Sistema: 2x - y = 7, x + 5y = -2
det A = 11
det Ax = 33 → x = 3
det Ay = -11 → y = -1
S = {(3, -1)}

Discussão de Sistemas Lineares

Um sistema linear pode ser:

  • SPD: uma única solução (retas se cruzam em um ponto).
  • SPI: infinitas soluções (retas coincidentes ou planos sobrepostos).
  • SI: nenhuma solução (retas paralelas ou planos que não se intersectam).
