📘 Matemática – Limites

O que é um limite?

O conceito de limite é um dos pilares do cálculo. Ele descreve o comportamento de uma função quando a variável se aproxima de um determinado ponto. Em outras palavras, o limite nos mostra qual valor uma função “tende” a alcançar, mesmo que ela nunca chegue exatamente a ele.

Exemplo intuitivo:
Imagine que você está caminhando em direção a uma parede. A cada passo, você percorre metade da distância restante. Você nunca toca exatamente a parede, mas se aproxima indefinidamente — o ponto em que você “chegaria” é o limite.

Definição simbólica

limx → a f(x) = L

Isso significa que, conforme x se aproxima de a, o valor de f(x) se aproxima de L. O limite pode existir mesmo que a função não esteja definida em a.

Regras básicas de limites

  • lim (c) = c   → O limite de uma constante é ela mesma.
  • lim (x) = a   → O limite de x quando x → a é o próprio a.
  • lim [f(x) + g(x)] = lim f(x) + lim g(x)
  • lim [f(x) · g(x)] = (lim f(x)) · (lim g(x))
  • lim [f(x)/g(x)] = (lim f(x)) / (lim g(x)), desde que o denominador ≠ 0

Exemplo prático

Calcule o limite:
limx → 2 (x² + 3x - 4)
Basta substituir:
= (2)² + 3·2 - 4 = 4 + 6 - 4 = 6
Portanto, lim(x → 2) (x² + 3x - 4) = 6.

Limites laterais

Às vezes, o limite pode ser diferente dependendo do lado pelo qual nos aproximamos. Nesse caso, temos o limite pela esquerda (x → a⁻) e o limite pela direita (x → a⁺).

Exemplo:
A função f(x) = |x| / x tem limites diferentes quando x → 0⁻ e x → 0⁺. Logo, o limite em x = 0 não existe.

🧮 Exercícios para praticar

  1. Calcule: limx → 3 (2x + 5)
  2. Determine o limite: limx → -1 (x² + 2x + 1)
  3. Existe o limite de f(x) = |x|/x em x = 0?
  4. Resolva: limx → 0 (sin x / x)
