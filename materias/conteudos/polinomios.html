📘 Matemática – Polinômios

O que são Polinômios?

Os polinômios são expressões algébricas formadas por números (coeficientes) e letras (partes literais). As letras representam variáveis, e cada combinação de número e letra é chamada de termo.

Exemplos:
a) 3ab + 5
b) x³ + 4xy - 2x²y³
c) 25x² - 9y²

Monômio, Binômio e Trinômio

Os polinômios são classificados de acordo com o número de termos:

  • Monômio: possui apenas um termo (ex: 3x, 5abc, x²y³z⁴)
  • Binômio: possui dois termos (ex: a² - b², 3x + y)
  • Trinômio: possui três termos (ex: x² + 3x + 7)

Grau dos Polinômios

O grau de um polinômio é determinado pela soma dos expoentes das variáveis no termo de maior grau.

Exemplo 1: 2x³ + y → grau = 3
Exemplo 2: 4x²y + 8x³y³ - xy⁴ → os graus são 3, 6 e 5 → grau = 6

Obs: o polinômio nulo (com todos os coeficientes iguais a zero) não possui grau definido.

Operações com Polinômios

Adição

Somamos os coeficientes dos termos semelhantes (mesma parte literal).

(-7x³ + 5x²y - xy + 4y) + (-2x²y + 8xy - 7y) = -7x³ + 3x²y + 7xy - 3y

Subtração

Invertem-se os sinais dentro dos parênteses e somam-se os termos semelhantes.

(4x² - 5xk + 6k) - (3xk - 8k) = 4x² - 8xk + 14k

Multiplicação

Multiplicamos termo a termo, somando os expoentes das letras iguais.

(3x² - 5x + 8)(-2x + 1) = -6x³ + 13x² - 21x + 8

Divisão

Utiliza-se o método da chave: divide-se coeficientes e subtrai-se os expoentes das mesmas bases.

Fatoração de Polinômios

Fatorar é escrever o polinômio como produto de fatores mais simples. Veja os principais casos:

  • Fator comum em evidência: ax + bx = x(a + b)
  • Agrupamento: ax + bx + ay + by = (x + y)(a + b)
  • Trinômio quadrado perfeito (adição): a² + 2ab + b² = (a + b)²
  • Trinômio quadrado perfeito (diferença): a² - 2ab + b² = (a - b)²
  • Diferença de dois quadrados: a² - b² = (a + b)(a - b)
  • Cubo perfeito (adição): a³ + 3a²b + 3ab² + b³ = (a + b)³
  • Cubo perfeito (diferença): a³ - 3a²b + 3ab² - b³ = (a - b)³
