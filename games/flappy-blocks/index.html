Tutorial: Crie seu Jogo!

Bem-vindo ao Flappy Bird Programável! Use os blocos para controlar o jogo.

  • Botão "Executar Blocos": Lê todos os seus blocos e inicia o jogo com essa lógica.
  • Botão "Reiniciar Jogo": Apenas reinicia o jogo, mantendo a lógica que você já executou.

Blocos Principais:

  • quando o jogo começar: Use este bloco para definir as "regras" do seu jogo, como a gravidade, a velocidade dos canos, etc.
  • quando clicar/tocar: Este é o bloco mais importante! Coloque o bloco bater asas aqui dentro.
  • quando passar por um cano: O que acontece when o pássaro pontua? Use o bloco adicionar ao placar aqui.
  • quando bater em um obstáculo: O que acontece when o pássaro bate? Use o bloco fim de jogo.

Combine os blocos e crie sua própria versão do Flappy Bird!
